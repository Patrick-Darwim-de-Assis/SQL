## Core Concepts Practiced
* **Subqueries**: Using inner queries to find maximum values dynamically, ensuring code handles changing data and empty tables (`NULL` cases).
* **Self-Joins**: Connecting a table to itself to compare records within the same dataset (like comparing an employee's salary directly to their manager's salary).
* **Filtering Scopes**: Correctly using `ON` and `WHERE` clauses to match and filter complex data relationships.
